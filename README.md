# Job Portal Web Application (Flask)

This is a beginner-friendly Job Portal web application built using Python Flask.

## Features
- Login page
- Recruiter can post jobs
- Users can view job listings
- SQLite database
- Flask backend

## Tech Stack
- Python
- Flask
- HTML
- SQLite

## How to Run
pip install -r requirements.txt
python app.py
Open http://127.0.0.1:5000/
